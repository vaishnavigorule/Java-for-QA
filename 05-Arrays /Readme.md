# Java Arrays

An array is a collection of elements of the same data type stored in contiguous memory locations.

## Topics Covered

- Array Declaration
- Array Initialization
- Array Traversal
- Largest Element
- Smallest Element
- Sum of Array
- Average of Array
- Reverse Array
- Copy Array
- Multi-Dimensional Array

## Why QA Engineers Learn Arrays?

Arrays are useful for:

- Storing test data
- Reading multiple values
- Data-driven testing
- Iterating through web elements
- Handling collections of similar data
