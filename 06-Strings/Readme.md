# Java Strings

A String is a sequence of characters used to store and manipulate text.

- String Creation
- String Methods

- StringBuilder
- StringBuffer

- Count Words
- Remove Spaces
- Duplicate Characters
- Anagram Check
- StringPrograms

## Why QA Engineers Learn Strings?

Strings are heavily used in Selenium for:

- Reading web element text
- Comparing expected vs actual values
- Handling URLs
- Working with test data
- Validating messages
