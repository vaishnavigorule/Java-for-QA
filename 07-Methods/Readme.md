# Java Methods

A method is a block of code that performs a specific task. Methods improve code reusability, readability, and maintainability.

## Topics Covered

- Method Without Parameters
- Method With Parameters
- Method With Return Type
- Method Without Return Type
- Static Methods
- Non-Static Methods
- Method Overloading
- Recursive Methods
- MethodPrograms

## Why QA Engineers Learn Methods?

Methods are used in Selenium Automation to:

- Reuse code
- Organize test scripts
- Create utility functions
- Implement Page Object Model (POM)
- Reduce code duplication
