# Java Basics

Java is a high-level, object-oriented programming language widely used for developing web applications, desktop software, mobile applications, and automation testing frameworks like Selenium.

## Topics Covered

- Java Program Structure
- Hello World Program
- Variables
- Data Types
- User Input
- Comments

## Why QA Engineers Learn Java?

Java is one of the most popular languages used with Selenium Automation Testing. Learning Java helps QA Engineers write automation scripts, understand application logic, and perform coding tasks during technical interviews.
