# Java for QA Engineers ☕

Welcome to the **Java-for-QA** repository!

This repository is designed for QA Engineers who want to build a strong foundation in Java before learning Selenium Automation Testing.

## 📚 Topics Covered

- Java Basics
- Variables & Data Types
- Operators
- Conditional Statements
- Loops
- Arrays
- Strings
- Methods
- Object-Oriented Programming (OOP)
- Constructors
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation
- Collections Framework
- Exception Handling
- File Handling
- Practice Programs
- Interview Questions
- Mini Projects

## 🎯 Why Java for QA?

Java is one of the most widely used programming languages in Selenium Automation Testing. Understanding Java helps QA Engineers write maintainable automation scripts and perform technical interviews confidently.
