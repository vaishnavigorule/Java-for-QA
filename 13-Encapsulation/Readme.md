# Java Encapsulation

Encapsulation is the process of wrapping data (variables) and methods into a single unit (class) while restricting direct access using private access modifiers.

## Topics Covered

- Private Variables
- Getter Methods
- Setter Methods
- Data Hiding
- Immutable Class

## Why QA Engineers Learn Encapsulation?

Encapsulation is widely used in Selenium:

- Page Object Model (POM)
- Utility Classes
- Configuration Classes
- Framework Design

Benefits:
- Data Security
- Better Maintainability
- Reusable Code
