# Java Abstraction

Abstraction is the process of hiding implementation details and showing only essential functionality.

## Topics Covered

- Abstract Class
- Abstract Method
- Interface
- Multiple Inheritance using Interface
- Default Methods
- Static Methods

## Why QA Engineers Learn Abstraction?

Abstraction is widely used in:

- Selenium WebDriver
- TestNG Framework
- Page Object Model
- Automation Framework Design

Example:
WebDriver driver = new ChromeDriver();

Here, WebDriver is an interface and ChromeDriver provides the implementation.
