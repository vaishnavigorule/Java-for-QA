# Java Operators

Operators are symbols used to perform operations on variables and values.

## Topics Covered

- Arithmetic Operators
- Relational Operators
- Logical Operators
- Assignment Operators
- Unary Operators
- Ternary Operator
- Bitwise Operators

## Why QA Engineers Learn Operators?

Operators are used in Selenium automation for writing conditions, validations, loops, assertions, calculations, and decision-making logic.
