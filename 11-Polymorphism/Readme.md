# Java Polymorphism

Polymorphism means "many forms". It allows the same method to perform different behaviors depending on the object.

## Topics Covered

- Method Overloading
- Method Overriding
- Compile-Time Polymorphism
- Runtime Polymorphism
- Dynamic Method Dispatch
- Upcasting
- Downcasting

## Why QA Engineers Learn Polymorphism?

Polymorphism is useful in:

- Selenium Framework Design
- Page Object Model (POM)
- Writing flexible and reusable automation code
- Runtime object handling
