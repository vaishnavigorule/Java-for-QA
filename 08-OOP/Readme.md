# Java Object-Oriented Programming (OOP)

Object-Oriented Programming (OOP) is a programming paradigm based on classes and objects.

## Topics Covered

- Class
- Object
- Object Creation
- Instance Variables
- Instance Methods
- this Keyword
- Real-world Examples

## Why QA Engineers Learn OOP?

OOP is the backbone of Selenium Automation.

It helps in:

- Page Object Model (POM)
- Framework Design
- Code Reusability
- Maintainability
- Scalability
